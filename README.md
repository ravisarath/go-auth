# auth-server

Login API

```
@Domain/auth/login
input = {
   "username":"ashley.akua",
   "password":"P@33w0rd!"
}

```

Logout API

```
@Domain/auth/logout

Send bearer token

```

Refresh API

```
@Domain/auth/token/refresh
{
    "refresh_token":"DEW!!!@"
}


```

Create User API

```
@Domain/auth/create
{
    "email": "WQ!!@gmail.com",
    "password": "P@33w0rd!",
    "username": "ASPL:",
    "company": "NAME",
    "group": "TYPE",
    "admin":true/false
```
